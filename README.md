# zen arcade

## Running Locally

Make sure you have Clojure installed.  Also, install the [Heroku Toolbelt](https://toolbelt.heroku.com/).

```sh
$ git clone https://github.com/kylog/zen-arcade.git
$ cd zen-arcade
$ lein repl
user=> (require 'zen-arcade.web)
user=>(def server (zen-arcade.web/-main))
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```sh
$ heroku create
$ git push heroku master
$ heroku open
```
